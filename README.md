# eshop_homepage

This repository contains the source code and documentation for the Eshop Home project,launched in November 2023.Eshop Home is an e-commmerce platform designed to provide a seamless shopping experience for users looking to purchase home goods and decor.

## Requirements

-HTML5
-CSS3
-JavaScript (ES6+)
-Tailwind CSS
-Font Awesome
-Responsive Design Principles
-Google Fonts

## Features

-User -friendly interface with intuitive navigation
-Responsive design for optimal viewing on all devices 
-Integration with Font Awesome for icons
-Customizable themes using Tailing CSS
-SEO-friendly structure for better search engine ranking
